
# HTML Image Tag Tutorial

## About the Project
This is an interactive HTML tutorial I coded to teach my classmates in middle school about HTML image tags! The project demonstrates how to use the <img> tag with all its important attributes and shows different border styling techniques.

## Live Demo

Check out the live tutorial at:  
**demo.sslabs.online/html-img-tag-tutorial/**

## Tip

Use the arrow keys to go forward, backward, and reset.
